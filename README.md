# 📝 Todo List App

A clean and functional Todo List application built with Next.js and Tailwind CSS.

## 🔴 Live Demo
[frontendnitin-todo.vercel.app](https://frontendnitin-todo.vercel.app)

## ✨ Features

- ➕ Add tasks with title and description
- ✏️ Edit tasks inline
- 🗑️ Delete tasks
- 💾 Persistent tasks using **Local Storage**
- 📭 Empty state message when no tasks available

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| Next.js 15 | React Framework |
| Tailwind CSS | Styling |
| Local Storage | Task persistence |
| React Hooks | State management |

## 🚀 Run Locally

**1. Clone the repository:**
```bash
git clone https://github.com/frontendnitin/todo-list-react.git
```

**2. Install dependencies:**
```bash
npm install
```

**3. Start development server:**
```bash
npm run dev
```

**4. Open in browser:**
```
http://localhost:3000
```
